The Unvired HTML5 Hybrid SDK is based on the Apache Cordova project and enables application developers to build native mobile applications in HTML5 using frameworks like Sencha Touch, JQuery, Ionic, SAP Open UI5 etc. Unvired HTML5 Hybrid SDK and Unvired Native SDK together provide complete choice to enterprises and application developers to build high quality enterprise mobile applications connecting to SAP, Oracle, Sharepoint etc easily.

Supported Mobile Platforms

  • Android
  • iOS
